# ansible-demo
ansible-demoは、Ansibleを使用してAWSのEC2インスタンス上でRailsの環境構築を自動で行います。  
# 使用技術
- Ansible
  - yum
  - Yarn
  - Node.js
  - Ruby
  - MySQL
  - Rails
  - Nginx
  - Unicorn
- AWS
  - VPC
  - EC2
